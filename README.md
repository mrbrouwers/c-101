# C 101 - Introductie

Welkom bij de mini-cursus **C 101**!

In deze GitHub-repository heb ik een aantal dingen voor jullie klaargezet om jullie hopelijk een goede kick-off te geven richting robotica deze periode. C en Arduino zijn erg soortgelijk, dus alles dat je hier leert zal straks zeker van pas komen.

Voor diegenen die eerder al geprogrammeerd hebben (bijvoorbeeld in Python) kan het zijn dat er bekende concepten voorbij komen. Let toch nog goed op! Tenzij je bekend bent met C++, Java of C# zijn er wel verschillen met andere talen, vooral in de syntax.

## Deze repository uitgelegd

In de repository zijn een aantal mappen te vinden. Deze bevatten allemaal een `.c` bestand. Elk mapje en bestand gaat over een ander concept binnen het programmeren. In elk bestand staat uitleg en een aantal oefeningen om dat concept mee onder de knie te krijgen.

We behandelen de volgende concepten:

1. Basis syntax (printf, main)
2. Functies
3. Datatypes
4. If-statements (voorwaarden)
5. Loops (herhaling)

Na deze mini-lessen kun je als het goed is:

- een C-bestand schrijven dat compileert
- tekst naar het scherm printen
- eenvoudige berekeningen uitvoeren
- beslissingen maken met `if`
- herhaling toepassen met `while` / `for`

## ⚠ Compileren

Dit is **belangrijk!** C verschilt met Python in de zin dat we voor een C-programma een **compiler** moeten gebruiken. Wat dit allemaal betekent is nog niet belangrijk, dus maak je daar geen zorgen om. Wat je moet weten:

1. Als je je code/programma hebt geschreven, moet je met je **terminal** de map van het betreffende `.c` bestand in door middel van `cd`. (bijv. `cd c-101/01_syntax`)
2. Je gebruikt de `gcc`-command als volgt: `gcc bestandsnaam.c -o programma`
3. Tot slot voer je `./programma` uit.
4. Je hebt je programma succesvol gecompileerd en uitgevoerd!

Hopelijk kunnen we in één les door alle stof heen komen! Zo niet is dat ook niet erg; probeer dan thuis of tijdens een ander geschikt moment de rest van de stof door te nemen.
